Zoho Books to Odoo Accounting Connector

Import your Zoho Books data into Odoo Accounting

This module allows you to import data from Zoho Books to Odoo Accounting.

Features

  • Import customers and vendors
  • Import chart of accounts
  • Import bills and invoices
  • Import journal entries
  • Configuration for API credentials
  • Mapping between Zoho Books and Odoo Accounting

How to use

  1. Configure your Zoho Books API credentials in the Configuration menu
  2. Set up mappings between Zoho Books and Odoo entities
  3. Use the import wizard to import data from Zoho Books
  4. View import history and logs to track the import process
